## [2.1.1] - 2026-03-25

### ✨ 新增

- **Cookie 获取功能扩展**
  - 支持 8 种不同的 Cookie 获取方法
  - 手动输入、浏览器自动提取、文件导入、EditThisCookie 导出、剪贴板识别等
  - 每种方法提供详细的操作步骤、适用场景和优缺点对比
- **反反爬虫机制**
  - 支持 10 个主流平台的反爬虫策略
  - User-Agent 随机化
  - 代理 IP 池配置
  - 请求间隔控制
  - 验证码自动识别集成
  - 模拟真实用户行为路径
  - 请求头轮换
- **可视化加载进程**
  - 动态加载进度显示（0-100%）
  - 阶段状态提示（验证Cookie、连接服务器、执行签到等）
  - 成功/失败状态的明确视觉反馈
  - 实时进度条更新
- **帮助中心优化**
  - 重构帮助中心内容结构
  - 覆盖所有功能模块
  - 详细的操作步骤
  - 常见问题解答（每个功能至少 5 个）
  - 错误处理方案（每个错误类型至少 3 种）
  - 实际使用案例
- **历史记录显示优化**
  - 仅显示用户实际执行过的签到记录
  - 未执行项目显示为灰色示例条目
  - 明确标注"未开始"状态
  - 历史记录筛选功能
  - 历史记录搜索功能
  - 历史记录导出功能

### 🔧 改进

- Web UI 界面全面优化
- 用户体验提升
- 响应式设计改进
- 错误提示更加友好
- 平台风险等级显示
- Cookie 获取方法说明更加详细

### 📚 文档

- 更新 README 文档
- 添加 Cookie 获取方法说明

## 版本说明

### 版本号格式

遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- `✨ 新增` - 新功能
- `🔧 改进` - 现有功能的改进
- `🐛 修复` - Bug 修复
- `🚀 性能` - 性能优化
- `📚 文档` - 文档更新
- `🎨 样式` - 代码格式调整
- `♻️ 重构` - 代码重构
- `🧪 测试` - 测试相关
- `🔧 配置` - 配置文件更新
- `⬆️ 依赖` - 依赖更新
