# Contributing to 996 Agent

First off, thank you for considering contributing! 🎉

## 🤝 How Can I Contribute?

### 🐛 Reporting Bugs
- Use the Bug Report template
- Include detailed reproduction steps
- What did the AI do wrong? Did it not involute hard enough? 😏

### 💡 Suggesting Enhancements
- Use the Feature Request template
- Be creative! More involution mechanisms welcome
- New specialized roles also welcome

### ✨ Submitting Pull Requests
1. Fork the repo
2. Create your feature branch
3. Make your changes
4. Ensure CI passes
5. Submit PR!

> 💡 **CTO says:** "Your PR better be good. If it's garbage, you're eliminated."

## 🌟 New Specialist Roles

Want to add a new team member? Great! Just make sure:
1. It fills a genuine gap in the current team
2. Has clear specialization, not overlapping with existing roles
3. Has meaningful KPIs and evaluation criteria
4. Brings genuinely new perspective
